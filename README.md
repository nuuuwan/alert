This is v0 of an "Awareness, Logistics, Evacuation, Recovery and Tracking" platform for natural disasters in Sri Lanka.

It aggregates 1) official alerts (DMC etc), 2) data-driven automatic alerts and 3) disaster related information.

The goal is to give people #autonomy during disasters, by providing timely information, so they can judge risk, see emerging danger, & act early, without over-dependence on authorities.

To use,

1. Select Location
2. View Alerts
3. View More Details
